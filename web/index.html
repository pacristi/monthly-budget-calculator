Patrones para detectar el sueldo

Substrings (sin distinción de mayúsculas) que el sistema usa para identificar tu depósito de sueldo. Si tu empleador escribe la descripción como "PAGO DE SUELDOS 12345", alcanza con poner pago de sueldos.

    Substrings que si aparecen en la descripción de un movimiento, lo descartan del cálculo del presupuesto. Útil para traspasos a cuentas propias, ahorros, pagos de TC, etc.
